---

**Modul 1.5: Reviewer-Agenten im Detail**

Das Marketing-Kit enthält 6 spezialisierte Reviewer-Agenten. Diese liefern Feedback auf Expertenniveau aus unterschiedlichen Perspektiven, um die Inhaltsqualität sicherzustellen.

**Dauer:** ~30 Minuten

---

### Schritt 1: Warum Reviewer-Agenten?

Jedes Marketingmaterial sollte aus mehreren Blickwinkeln überprüft werden:
- Markenkonsistenz
- SEO-Optimierung
- Conversion-Potenzial
- Persona-Passung
- Praktische Umsetzbarkeit

Reviewer-Agenten geben dir Experten auf Abruf für jeden Bereich.

### Schritt 2: Die Experten-Reviewer

Erkläre die drei Experten-Reviewer:

**`brand-voice-guardian`:**
- Stellt Markenstimmen-Konsistenz sicher
- Überprüft Tonalität und Botschaften
- Validiert gegen Markenrichtlinien
- Kennzeichnet markenuntypische Inhalte

**`seo-specialist`:**
- Bewertet Keyword-Verwendung
- Überprüft On-Page-SEO-Elemente
- Prüft Inhaltsstruktur
- Identifiziert Optimierungsmöglichkeiten

**`conversion-optimizer`:**
- Bewertet CTA-Effektivität
- Überprüft Überzeugungselemente
- Identifiziert Reibungspunkte
- Schlägt Conversion-Verbesserungen vor

### Schritt 3: Die Persona-Reviewer

Erkläre die drei Persona-Reviewer:

**`manager-maria` (Marketing Manager, 38 Jahre):**
- B2B-Mittelstandsunternehmen-Perspektive
- Bewertet Umsetzbarkeit im Team
- Berücksichtigt Ressourcenbeschränkungen
- Prüft Kampagnen-Durchführbarkeit

**`solo-steve` (Solopreneur, 32 Jahre):**
- Ein-Personen-Unternehmen-Perspektive
- Bewertet Zeiteffizienz
- Prüft Self-Service-Einfachheit
- Berücksichtigt Budget-Sensibilität

**`startup-sam` (Startup-Gründer, 28 Jahre):**
- Frühphasen-Startup-Perspektive
- Bewertet Wachstumspotenzial
- Prüft Viralitätsfaktoren
- Bewertet Time-to-Market

### Schritt 4: Experten-Reviewer einsetzen

Übe mit Experten-Reviewern:

```
Review our AgentKits blog post with expert reviewers:

1. `brand-voice-guardian`: Evaluate brand consistency
2. `seo-specialist`: Evaluate SEO optimization
3. `conversion-optimizer`: Evaluate conversion potential

Each should score (1-10) and provide:
- Top 3 strengths
- Top 3 improvements needed
- Specific recommended edits
```

### Schritt 5: Persona-Reviewer einsetzen

Übe mit Persona-Reviewern:

```
Review our AgentKits email sequence from persona perspectives:

1. `manager-maria`: Would a B2B marketing manager find this valuable?
2. `solo-steve`: Would a solopreneur implement this?
3. `startup-sam`: Would this drive growth for a startup?

Each should evaluate:
- Resonance with their needs
- Implementation feasibility
- Value perception
```

### Schritt 6: Kombiniertes Review-Gremium

Nutze alle Reviewer zusammen für umfassendes Feedback:

```
Run a full review board on the AgentKits landing page copy:

Expert Reviews:
- Brand consistency (brand-voice-guardian)
- SEO optimization (seo-specialist)
- Conversion potential (conversion-optimizer)

Persona Reviews:
- B2B fit (manager-maria)
- Self-service potential (solo-steve)
- Growth potential (startup-sam)

Synthesize: What changes would satisfy all reviewers?
```

### Schritt 7: Iterative Verbesserung

Zeige den Review-Verbesserungs-Zyklus:

```
1. Create content
2. Get multi-reviewer feedback
3. Implement top priority changes
4. Re-review changed content
5. Repeat until all reviewers score 8+
```

### Schritt 8: Wann welchen Reviewer einsetzen

Gib Orientierung:

| Inhaltstyp | Primäre Reviewer |
|-------------|-------------------|
| Blog-Posts | seo-specialist, brand-voice-guardian |
| Landing Pages | conversion-optimizer, manager-maria |
| E-Mail-Sequenzen | email-wizard, startup-sam |
| Social-Media-Inhalte | brand-voice-guardian, solo-steve |
| Verkaufsunterlagen | sales-enabler, manager-maria |
