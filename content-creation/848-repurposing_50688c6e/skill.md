# Content Repurposing

Maximizing content value through repurposing.

## Repurposing Framework

### One-to-Many Approach
One piece of content → Multiple formats and channels

### Content Multiplication Example
**Starting Asset: 60-minute webinar**

1. Full recording → YouTube
2. Audio → Podcast episode
3. Transcript → Blog post
4. Key points → Twitter thread
5. Quotes → Social graphics
6. Slides → SlideShare
7. Q&A → FAQ content
8. Clips → Short-form video
9. Summary → Email newsletter
10. Tips → Instagram carousel

## Best Practices

### Content Quality
- Start with comprehensive source
- Adapt for format (don't just copy)
- Optimize for each platform
- Update dated information

### Timing
- Launch core asset first
- Roll out repurposed content over weeks
- Space distribution strategically
- Maintain freshness

### Tracking
- Tag by original source
- Track aggregate performance
- Identify best-performing formats
- Optimize repurposing strategy
