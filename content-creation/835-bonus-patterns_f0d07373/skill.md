## Pattern Library Là Gì?

Pattern library ghi lại:
- **Winning patterns** - Điều gì hoạt động và tại sao
- **Failing patterns** - Điều gì không hoạt động và tại sao
- **Persona-specific patterns** - Điều gì cộng hưởng với từng audience
- **Channel-specific patterns** - Điều gì hoạt động trên từng platform
- **Seasonal patterns** - Điều gì hoạt động ở các thời điểm khác nhau

## Headline Patterns

### High-Performers (2x+ trung bình)
| Pattern | Ví Dụ | Performance | Context |
|---------|---------|-------------|---------|
| Cách [Persona] có thể [Benefit] | "Cách quản lý remote tiết kiệm 10 giờ/tuần" | 2.3x CTR | Hoạt động tốt nhất cho Solo Sam |
| [Number] cách để [Outcome] | "7 cách phối hợp focus của team" | 2.1x CTR | Số lẻ outperform số chẵn |
| Hướng dẫn [Adjective] về [Topic] | "Hướng dẫn hoàn chỉnh về năng suất remote" | 1.8x CTR | Tốt nhất cho pillar content |

### Underperformers (Tránh)
| Pattern | Tại Sao Thất Bại |
|---------|---------------|
| "Revolutionary" anything | Quá hyperbolic, trust thấp |
| Clickbait questions | Opens cao, engagement thấp |

## Email Patterns

### Subject Lines
| Pattern | Open Rate | Notes |
|---------|-----------|-------|
| [First Name], [question]? | +34% | Personalization + curiosity |
| [Number] inside | +28% | Tạo anticipation |
| Quick [benefit] | +22% | Tôn trọng thời gian |

### Body Copy
- P.S. lines tăng clicks 22%
- Single CTA outperform multiple 3:1
- First paragraph dưới 2 sentences

### Send Times
| Ngày | Giờ | Performance |
|-----|------|-------------|
| Thứ Ba | 10:00 AM | Tốt nhất overall |
| Thứ Năm | 2:00 PM | Thứ hai tốt nhất |
| Thứ Hai AM | - | -40% engagement |

## Persona-Specific Patterns

### Solo Sam (B2B Technical)
- Phản hồi tốt với: ROI, efficiency, metrics
- Tránh: Fluffy language, vague benefits
- Best channels: LinkedIn, Email
- Best times: Thứ Ba-Thứ Năm, business hours

### Startup Sam (Founder)
- Phản hồi tốt với: Growth, scale, speed
- Tránh: Enterprise jargon
- Best channels: Twitter, Product Hunt
- Best times: Tối, cuối tuần

### Solo Steve (Solopreneur)
- Phản hồi tốt với: Time savings, DIY, affordability
- Tránh: Team-focused messaging
- Best channels: Twitter, Email
- Best times: Sáng sớm, tối

## Pattern Library Best Practices

1. **Review trước campaigns** - Kiểm tra điều gì đã hoạt động trước
2. **Update sau campaigns** - Thêm learnings mới
3. **Include failures** - Biết điều gì nên tránh
4. **Add context** - Tại sao nó hoạt động quan trọng
5. **Track metrics** - Quantify performance

## CRO Patterns

### Landing Page Patterns

| Element | Winning Pattern | Improvement |
|---------|-----------------|-------------|
| Headline | Specific benefit + number | +40% engagement |
| Subheadline | "How" hoặc "Who it's for" | +25% clarity |
| CTA | First-person ("Bắt Đầu Trial Của Tôi") | +28% clicks |
| Form fields | 3-4 fields max | +35% completions |
| Social proof | Logos + "Join X teams" | +22% trust |

### Form Patterns

| Pattern | Impact |
|---------|--------|
| Single column layout | +15% completion |
| Progress indicator (multi-step) | +20% completion |
| Inline validation | +22% reduction in errors |
| Smart defaults | +10% completion |
| "Không cần thẻ tín dụng" | +18% signups |

### Popup Patterns

| Type | Timing | Conversion |
|------|--------|------------|
| Exit intent | Mouse rời viewport | 3-5% capture |
| Scroll 50% | Engaged visitors | 2-4% capture |
| Time delay 30s | Sau reading | 1-3% capture |

**High-Converting Popup Elements:**
- Clear value trong headline (không phải "Subscribe")
- Single field (email only)
- Visible close button
- Mobile-friendly (full-screen OK)
- Frequency cap (once per session)

### Signup Flow Patterns

| Pattern | Conversion Impact |
|---------|-------------------|
| Email-only start | +45% initial capture |
| Google/SSO option | +30% cho consumer |
| Progressive profiling | +25% vs long form |
| Show password option | +12% completions |

### Onboarding Patterns

| Element | Activation Impact |
|---------|-------------------|
| Empty state với value | +35% action rate |
| Sample data/templates | +28% exploration |
| 3-step checklist | +40% completion |
| Aha moment trong <5 phút | +50% trial conversion |

### Paywall Patterns

| Trigger | Conversion |
|---------|------------|
| After value moment | Cao nhất (2-4%) |
| Feature gate | Medium (1-2%) |
| Usage limit | Variable |
| Trial expiry | Phụ thuộc engagement |

**Effective Upgrade Screen Elements:**
- Feature comparison table
- Current plan highlighted
- Testimonial từ similar user
- "Most popular" badge
- Annual discount prominent
