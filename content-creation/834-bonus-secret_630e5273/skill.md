## Die 10x Marketer Denkweise

Die meisten Marketer arbeiten **linear** - jede Kampagne erfordert denselben Aufwand.

**10x Marketer** arbeiten **exponentiell** - jede Kampagne macht die nächste schneller und besser.

Das Geheimnis: **Systematische Akkumulation**

---

## Die vier Säulen

### Säule 1: Befehlsbeherrschung

Wisse, welche Befehle du miteinander verketten solltest:

**Kampagnenstart-Workflow:**
```
1. /research:market → Markt verstehen
2. /research:persona → Zielgruppe definieren
3. /campaign:plan → Strategischer Plan
4. /campaign:brief → Kreative Ausrichtung
5. /campaign:calendar → Content-Zeitplan
6. /seo:keywords → SEO-Fundament
7. /content:blog → Haupt-Content
8. /content:social → Distributions-Content
9. /sequence:welcome → Lead-Pflege
10. /analytics:funnel → Messung
```

**Content-Produktions-Workflow:**
```
1. /seo:keywords → Chancen finden
2. /content:blog → Content erstellen
3. /seo:optimize → Für Suchmaschinen optimieren
4. /content:social → Für Social Media aufbereiten
5. Review with agents → Qualitätsprüfung
```

### Säule 2: Agenten-Orchestrierung

Nutze Agenten strategisch:

**Für die Recherchephase:**
- `researcher` → Marktverständnis
- `attraction-specialist` → Wettbewerbsanalyse

**Für die Erstellungsphase:**
- `brainstormer` → Kampagnenkonzepte
- `copywriter` → Content-Erstellung
- `planner` → Strategische Struktur

**Für die Review-Phase:**
- `brand-voice-guardian` → Stimmenprüfung
- `conversion-optimizer` → CRO-Review
- `seo-specialist` → SEO-Validierung
- Persona-Reviewer → Zielgruppen-Fit

**Für die Optimierungsphase:**
- `lead-qualifier` → Lead-Qualität
- `email-wizard` → Sequenz-Optimierung
- `continuity-specialist` → Kundenbindung

### Säule 3: Dokumentations-Leverage

Nutze Dokumentation, um Wissen zu akkumulieren:

**Vor jeder Kampagne:**
```
1. Lies docs/campaign-playbooks.md
2. Lies docs/content-style-guide.md
3. Lies docs/channel-strategies.md
4. Überprüfe deine Musterbibliothek
```

**Nach jeder Kampagne:**
```
1. Aktualisiere Musterbibliothek mit Erkenntnissen
2. Dokumentiere erfolgreiche Ansätze
3. Notiere, was zu vermeiden ist
4. Verfolge Performance-Trends
```

### Säule 4: Die Akkumulationsschleife

## Das 10x Ausführungssystem

### Täglich: Quick Wins (15 Min.)
```
/ops:daily "focus area"
```
- Metriken prüfen
- 1 Content-Stück erstellen
- Musterbibliothek aktualisieren

### Wöchentlich: Strategische Überprüfung (1 Stunde)
```
/ops:weekly "current week"
```
- Kampagnen-Performance überprüfen
- Content der nächsten Woche planen
- Sequenzen basierend auf Daten aktualisieren

### Monatlich: Tiefenanalyse (2 Stunden)
```
/ops:monthly "current month"
/analytics:report "all campaigns" "full analysis"
```
- Umfassende Performance-Überprüfung
- Dokumentation aktualisieren
- Strategien verfeinern
