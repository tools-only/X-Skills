## الركائز الأربع

### الركيزة 1: إتقان الأوامر

اعرف أي الأوامر يجب ربطها معًا:

**سير عمل إطلاق الحملة:**
```
1. /research:market → فهم السوق
2. /research:persona → تحديد الجمهور
3. /campaign:plan → خطة استراتيجية
4. /campaign:brief → التوجيه الإبداعي
5. /campaign:calendar → جدول المحتوى
6. /seo:keywords → أساس تحسين محركات البحث
7. /content:blog → محتوى أساسي
8. /content:social → محتوى التوزيع
9. /sequence:welcome → رعاية العملاء المحتملين
10. /analytics:funnel → القياس
```

**سير عمل إنتاج المحتوى:**
```
1. /seo:keywords → إيجاد الفرص
2. /content:blog → إنشاء المحتوى
3. /seo:optimize → التحسين للبحث
4. /content:social → إعادة التوظيف لوسائل التواصل الاجتماعي
5. Review with agents → فحص الجودة
```

### الركيزة 2: تنسيق الوكلاء

استخدم الوكلاء بشكل استراتيجي:

**لمرحلة البحث:**
- `researcher` → فهم السوق
- `attraction-specialist` → المعلومات الاستخباراتية التنافسية

**لمرحلة الإنشاء:**
- `brainstormer` → مفاهيم الحملة
- `copywriter` → إنشاء المحتوى
- `planner` → البنية الاستراتيجية

**لمرحلة المراجعة:**
- `brand-voice-guardian` → فحص الصوت
- `conversion-optimizer` → مراجعة تحسين التحويل
- `seo-specialist` → التحقق من تحسين محركات البحث
- مراجعو الشخصيات → ملاءمة الجمهور

**لمرحلة التحسين:**
- `lead-qualifier` → جودة العملاء المحتملين
- `email-wizard` → تحسين التسلسل
- `continuity-specialist` → الاحتفاظ بالعملاء

## نظام التنفيذ الخارق (10x)

### يومي: انتصارات سريعة (15 دقيقة)
```
/ops:daily "focus area"
```
- فحص المقاييس
- إنشاء قطعة واحدة من المحتوى
- تحديث مكتبة الأنماط

### أسبوعي: مراجعة استراتيجية (ساعة واحدة)
```
/ops:weekly "current week"
```
- مراجعة أداء الحملة
- التخطيط لمحتوى الأسبوع القادم
- تحديث التسلسلات بناءً على البيانات

### شهري: تحليل عميق (ساعتان)
```
/ops:monthly "current month"
/analytics:report "all campaigns" "full analysis"
```
- مراجعة شاملة للأداء
- تحديث التوثيق
- تحسين الاستراتيجيات
