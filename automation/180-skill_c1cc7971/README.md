## Description

FluidSim is an objectoriented Python framework for highperformance computational fluid dynamics (CFD) simulations. It provides solvers for periodicdomain equations using pseudospectral methods with FFT, delivering performance comparable to Fortran/C++ while maintaining Python's ease of use.
