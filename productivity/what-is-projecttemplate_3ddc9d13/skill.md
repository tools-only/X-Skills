- Impact = (Revenue Impact x Time to Cash x Margin) / 3
- Optimizes for: profitability, cash flow, fast decisions

**VENTURE** -- Growth-funded. Market-share-first. Investor-backed.
- Impact = (Market Size x Defensibility x Growth Rate) / 3
- Optimizes for: growth rate, market size, defensibility

## Getting Started

1. Run `fnd-architect` to set mode and strategic context (populates 00-02)
2. Run `fnd-researcher` for discovery (populates 03-06)
3. Run `fnd-modeler` for business model (populates 07-09, 12-14)
4. Run `fnd-launch-planner` for execution (populates 10, 11, 15)
5. Define goals with `sys-defining-goals`
6. Decompose with `sys-decomposing-goals`
7. Activate threads with `sys-activating-goals`

Or run `fnd-builder` for the full pipeline end-to-end.
