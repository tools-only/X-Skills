## Description

模型上下文协议（Model Context Protocol，MCP） 是由Anthropic公司提出并开源的一项开放标准协议，旨在标准化大型语言模型与外部数据源和工具之间的交互方式。MCP被誉为AI应用的"USBC接口"，正如USBC为设备连接外设提供了标准化方式一样，MCP为AI模型连接不同的数据源和工具提供了统一的标准接口。
