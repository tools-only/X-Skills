## Overview

This t-shirt design showcases the powerful workflow of Claude Code Skills for creating intelligent textbooks. The simple three-box diagram illustrates the transformation from a basic course description to a comprehensive 500-page textbook.

## Design Concept

The workflow diagram uses a left-to-right flow perfect for t-shirt placement:

1. **Course Description** (Turquoise) - The starting point: a simple course outline
2. **Claude Code Skills** (Anthropic Brown) - The transformation engine: AI-powered automation
3. **500 Page Textbook** (Purple) - The final product: a complete educational resource

## Design Specifications

### Colors
- **Input Node** (Course Description): `#4ECDC4` - Vibrant turquoise
- **Process Node** (Claude Code Skills): `#DA7857` - Anthropic signature brown
- **Output Node** (500 Page Textbook): `#6C5CE7` - Bold purple

### Typography
- **Font Size**: 20px bold for maximum readability on fabric
- **Stroke Width**: 3px borders for clear definition
- **Arrow Labels**: 18px bold ("Transform" and "Generate")

### Layout
- **Direction**: Left-to-right (LR) horizontal flow
- **Recommended Placement**: Center chest or upper back
- **Best Fabric Colors**: White, light gray, cream, or light blue

## Printing Recommendations

### For Custom T-Shirt Services

1. **Download the design** using the "Download as SVG" button in the interactive version
2. **File format**: SVG (vector) for best quality at any size
3. **Alternative**: High-resolution PNG screenshot (2x or 3x scale)

### Recommended Print Services

- **Printful**: Upload SVG directly, choose "DTG" (Direct to Garment) printing
- **CustomInk**: Works well with PNG exports, bulk discounts available
- **Redbubble**: Artist-friendly, supports SVG uploads
- **TeeSpring**: Easy design upload, no minimum order quantity

### T-Shirt Styles That Work Well

- **Classic crew neck** (Gildan, Bella+Canvas)
- **Premium fitted** (Next Level, American Apparel)
- **Organic cotton** (Stanley/Stella, Alternative Apparel)
- **Technical/performance** (Under Armour, Nike Dri-FIT)

### Sizing Recommendations

- **Small/Medium designs**: 8-10 inches wide (portable, subtle)
- **Large designs**: 12-14 inches wide (statement piece, maximum visibility)
- **This design works best**: 10-12 inches wide for balanced visibility

## Variations to Consider

### Alternative Color Schemes

**Dark Mode** (for dark-colored shirts):
- Light text on dark background
- Neon accents (bright cyan, orange, purple)

**Monochrome** (classic black on white):
- Single color with gradient shading
- Professional, timeless look

**Anthropic Branded**:
- All boxes in shades of Anthropic brown (#DA7857)
- Gradient from light to dark

### Layout Variations
