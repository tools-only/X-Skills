## Description

The pdfviewer/ directory contains a Chrome extension implementing Mozilla's PDF.js library for inbrowser PDF rendering. This extension loads through browser_guard.py to enable PDF viewing without external applications. It intercepts PDF URLs and renders them using HTML5 Canvas and JavaScript.
