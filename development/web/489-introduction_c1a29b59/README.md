## Description

OpenViking 是一个开源的、专为 AI Agent 设计的上下文数据库。OpenViking 通过文件系统范式统一管理Agent 所需要的上下文（记忆、资源和技能），并实现上下文的分层供给与自我迭代，最终目标是降低 Agent 开发门槛，让开发者更专注于业务创新而非底层上下文管理。
