## Description

Use DataFrames when:
 Processing structured or semistructured data (JSON, Parquet, CSV, Avro)
 Performing SQLlike operations (joins, aggregations, filters)
 Need Catalyst optimizer benefits (predicate pushdown, column pruning)
 Working with columnar formats for better compression
