## Description

Static Token Auth allows Registry API endpoints (/api/, /v0.1/) to be accessed using a static API key instead of IdPbased JWT validation. This is useful in trusted network environments where configuring a full identity provider (Keycloak, Entra ID, Cognito) is not practical.
