## Description

This patch release addresses MongoDB authentication compatibility issues between MongoDB Community Edition and AWS DocumentDB. The changes enable the MCP Gateway Registry to work seamlessly with both MongoDB CE 8.2+ (using SCRAMSHA256) and AWS DocumentDB v5.0 (using SCRAMSHA1).
