## Description

OpenViking 目前主要以 Python SDK 形式提供服务，支持两种部署模式：
 Embedded 模式：自动启动本地 AGFS 子进程和 VikingVectorIndex（单例）
 Service 模式：连接远程 AGFS 和 VectorDB 服务（非单例）
