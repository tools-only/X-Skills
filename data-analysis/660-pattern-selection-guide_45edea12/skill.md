## Pattern Selection Guide

**Choose Perspective Multiplication if you:**

- Need diverse viewpoints on same problem
- Want to break out of existing mental models
- Serve multiple stakeholder groups
- Want 15-25 ideas from single brainstorming session

**Choose Constraint Variation if you:**

- Have resource limitations (budget, time, team)
- Want implementable ideas (not blue-sky thinking)
- Need ideas ranked by feasibility
- Want creativity to emerge from limitations

**Choose Inversion (Worst Possible Idea) if you:**

- Need to identify risks and failure modes
- Want to challenge assumptions
- Are designing for robustness/resilience
- Need to convince skeptical stakeholders

**Choose Analogical Transfer if you:**

- Want breakthrough innovation (30-40% higher novelty)
- Need cross-industry perspective
- Have time for deeper brainstorming session
- Want ideas that competitors haven't considered

**Choose Scenario/Time-Based if you:**

- Making strategic decisions with long-term implications
- Need alignment on future direction
- Want to identify inflection points
- Need to future-proof decisions
