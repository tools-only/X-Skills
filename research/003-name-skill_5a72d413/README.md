## Description

The Sepsis3 definition (2016) identifies sepsis as lifethreatening organ dysfunction caused by a dysregulated host response to infection. Operationalized as:
 Suspected infection (antibiotics + culture within time window) AND
 SOFA score >= 2 (within 48h before to 24h after suspected infection)
