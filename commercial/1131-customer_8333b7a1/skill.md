### Best Practices

- **3-5 personas max** - More = less focus
- **Data-driven** - Based on research, not assumptions
- **Update quarterly** - Customers evolve
- **Share widely** - Everyone should know personas

## Customer Journey Map

**Use when:** Improving customer experience, identifying pain points, optimizing touchpoints

### Structure

```
Stages:    AWARENESS ‚Üí CONSIDERATION ‚Üí PURCHASE ‚Üí USE ‚Üí LOYALTY
              ‚îÇ             ‚îÇ            ‚îÇ        ‚îÇ        ‚îÇ

Pain Points: [Frustrations/obstacles]
              ‚îÇ             ‚îÇ            ‚îÇ        ‚îÇ        ‚îÇ

```

### Template

| Stage | Awareness | Consideration | Purchase | Use | Loyalty |
|-------|-----------|---------------|----------|-----|---------|
| **Actions** | Searches for solutions | Compares options | Makes decision | Uses product | Recommends/renews |
| **Touchpoints** | Google, social media | Website, reviews | Checkout, payment | Product, support | Email, community |
| **Emotions** | Curious | Uncertain | Excited/Anxious | Frustrated/Delighted | Loyal/Indifferent |
| **Pain Points** | Hard to find info | Confusing pricing | Complex checkout | Poor onboarding | No recognition |
| **Opportunities** | SEO, content marketing | Clear comparison | Simplify flow | Better tutorials | Loyalty program |

### Journey Mapping Process

1. **Define scope** - Which persona? Which journey?
2. **Research** - Interviews, surveys, analytics
3. **Map current state** - How it works today
4. **Identify pain points** - Where is friction?
5. **Design future state** - How it should work
6. **Prioritize improvements** - Impact vs effort

## STP (Segmentation, Targeting, Positioning)

**Use when:** Market strategy, defining target market, competitive positioning

### The Three Steps

```
Segmentation ‚îÄ‚îÄ‚ñ∫ Targeting ‚îÄ‚îÄ‚ñ∫ Positioning
(Divide market)   (Choose       (Differentiate)
                  segments)
```

### 1. Segmentation Variables

| Type | Variables |
|------|-----------|
| **Demographic** | Age, gender, income, education, occupation |
| **Geographic** | Country, region, city, climate |
| **Psychographic** | Lifestyle, values, personality, interests |
| **Behavioral** | Usage rate, loyalty, benefits sought, occasions |

### 2. Targeting Strategies

| Strategy | Description | When to Use |
|----------|-------------|-------------|
| **Undifferentiated** | Same offering for all | Commodity products |
| **Differentiated** | Different offerings for different segments | Resources available |
| **Concentrated** | Focus on one segment | Limited resources, niche |
| **Micromarketing** | Individual-level customization | High-value customers |

### 3. Positioning Statement Template

```
For [target segment]
Who [need/want]
[Brand] is a [category]
That [key benefit]
Unlike [competitors]
We [differentiator]
```

**Example:**
```
For busy Thai professionals
Who want to advance their careers with better data skills
ThepExcel is an online learning platform
That provides practical, job-focused Excel and Power BI training
Unlike generic international courses
We teach in Thai with examples from Thai business contexts
```

## Marketing Mix (4Ps)

**Use when:** Marketing planning, product launch, competitive strategy

### The Four Ps

| P | Questions |
|---|-----------|
| **Product** | What are we selling? Features? Quality? Branding? |
| **Price** | How much? Discounts? Payment terms? Perceived value? |
| **Place** | Where sold? Distribution channels? Accessibility? |
| **Promotion** | How do customers learn about it? Advertising? PR? |

### Extended 7Ps (for Services)

| P | Meaning |
|---|---------|
| **People** | Staff who deliver service |
| **Process** | How service is delivered |
| **Physical Evidence** | Tangible elements of service |

## Jobs to Be Done (JTBD)

**Use when:** Product innovation, understanding true customer needs

### Core Concept

Customers don't buy products‚Äîthey "hire" them to get a job done.

### Job Statement Format

```
When [situation/trigger]
I want to [motivation]
So I can [expected outcome]
```

**Example:**
```
When I'm preparing a monthly report for my boss
I want to quickly summarize data from multiple sources
So I can present insights without spending hours on formatting
```

### Types of Jobs

| Type | Description | Example |
|------|-------------|---------|
| **Functional** | Practical task to complete | "Analyze sales data" |
| **Emotional** | How they want to feel | "Feel confident in meetings" |
| **Social** | How they want to be perceived | "Be seen as tech-savvy" |
